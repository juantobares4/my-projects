¿Qué es la Zona de Fresnel?


La Zona de Fresnel es un área alrededor del camino directo entre dos antenas de radio que necesita estar libre de obstáculos para asegurar una buena calidad de señal. Imaginá que la señal viaja como una cuerda tensa entre las antenas, y alrededor de esta cuerda hay un tubo invisible. Si hay algo en este tubo, como un árbol o un edificio, la señal puede rebotar y llegar débil o distorsionada. Mantener esta área despejada permite que la señal llegue fuerte y clara, lo que es crucial para una comunicación efectiva entre las antenas.
